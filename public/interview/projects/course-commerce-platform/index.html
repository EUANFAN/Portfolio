01Overview

What it is, in one sentence

A multi-environment mobile course commerce platform that unified course listing, payment, and order flows across WeChat, Alipay and H5 browsers — from fragmented legacy systems into a single component-driven codebase used by the company's core sales team.

Situation
Course sales ran across three runtimes (WeChat, Alipay, browser) with separate codebases, inconsistent UI, and duplicated business logic — every feature took 3x the effort.
Task
Unify the mobile commerce surface into a single frontend foundation without disrupting the running business during the transition.
Action
Built a cross-environment component library and delivery pipeline; re-platformed the core course-to-payment flow; rolled out incrementally behind feature flags.
Result
Core transaction flow stable at millions-user scale; feature delivery time dropped meaningfully; UI & payment logic consistent across runtimes.
